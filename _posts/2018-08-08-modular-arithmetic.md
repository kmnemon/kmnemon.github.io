* Modular Arithmetic  
  
Definition Modulo Operation:  
Let a, r, m belong to Z(where Z is a set of all integers) and m>0. We write  
a = r mod m  
if m divides a - r.
m is called the modulus and r is called the remainder  
  
Computation of the Remainder  
It is always possible to write a belong to Z, such that  
a = q * m + r for 0 <= r < m  
  
* Integer Rings  
  
Definition Ring  
The integer ring Zm consisits of:  
1. The set Zm = {0,1,2,...., m-1}  
2. Two operations "+" and " * " for all a, b belong to Zm such that:  
a + b = c mod m, (c belong to Zm)  
a * b = d mod m, (d belong to Zm)
